# Sistema de Reservas de Hotel

Este repositorio contiene el diseño y los scripts de la base de datos para un sistema de hotel, correspondiente a la Práctica Guided 1. 

El modelo se pensó para conectar clientes, las habitaciones que reservan, los empleados que gestionan el proceso y el registro de los pagos.

## Tablas del proyecto:
* **habitaciones:** Datos de los cuartos (número, tipo, precio y si está disponible).
* **clientes:** Información de contacto de los huéspedes.
* **empleados:** Personal del hotel.
* **reservas:** Controla qué cliente se queda en qué habitación y quién lo atendió.
* **pagos:** Guarda el monto, fecha y método de pago de cada reserva.
